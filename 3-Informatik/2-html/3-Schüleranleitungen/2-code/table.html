Tabellen

Tabellen werden mit folgendem eingefügt:
Beispiel(Quellcode vom Bild):
Üb 1.Sp Üb 2.Sp Üb 3.Sp
1.Zeile, 1.Spalte 1.Zeile, 2.Spalte 1.Zeile, 3.Spalte
2.Zeile, 1.Spalte 2.Zeile, 2.Spalte 2.Zeile, 3.Spalte


Dabei markiert "table" für den Beginn bzw das ende der Tabelle.
Mit dem "tr" Tag wird eine neue Zeile angewählt, in welcher dann mit dem "td" Tag jede Spalte nacheinander angewählt wird.
Der "th" Tag markiert abei in der eresten Zeile die Überschrift.
